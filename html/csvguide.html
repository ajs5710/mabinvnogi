You can use a custom URL to source the data in the Dye Visualizer.

Doing so requires a valid URL.

Probably the easiest way to do this is to use a google sheet (sheets.google.com).

Create your sheet and then go to File -> Share -> Publish to web

Then select the tab with your data and "Comma-seperated values" from the dropdowns.

This will give you a link that you can paste directly in the text box in the Dye Visualizer page.

Here is an example sheet you can use for reference.

You should be able to use any columns as long as the first column is Color.

The Type column and Pet columns are just recommended but could be anything really.

The Type column is just meant to communicated the type of Dye that it is.

While the Pet column is just meant to help you stay organized.

Only the first three columns are used by the page so feel free to add additional columns.
